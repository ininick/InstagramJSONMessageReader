# InstagramJSONMessageReader
This is an simple instagram message reader for downloaded data from instagram in JSON format. Only for message.

Step:
1. Download data from: Instagram -> More -> Your Activity -> Download Your Information -> JSON
2. After download, open the path from the message that you want to read or convert to txt
   ![image](https://github.com/ininick/InstagramJSONMessageReader/assets/110322459/6dc719da-3abb-4a55-857d-646beb593950)
3. Copy the path and paste to the program
   ![image](https://github.com/ininick/InstagramJSONMessageReader/assets/110322459/7f46c421-cac2-4f7c-9cc6-d236ac0ee55d)
4. Assume use the default, enter (keep blank)
   ![image](https://github.com/ininick/InstagramJSONMessageReader/assets/110322459/c30e1fe6-5287-48cd-8daf-287902745d52)
5. Enter oldest -> newest or vice versa
   ![image](https://github.com/ininick/InstagramJSONMessageReader/assets/110322459/f90f3813-e8dd-4f62-b0d5-a23271669c5f)
6. Then enter the output filename, default is message_1_result
   ![image](https://github.com/ininick/InstagramJSONMessageReader/assets/110322459/8ca0bed8-f616-4d33-9b26-0348a8e35315)
7. Then the result will appear as new .txt file
![image](https://github.com/ininick/InstagramJSONMessageReader/assets/110322459/2640c1be-f978-4fa9-bda5-8cb07c8a26f7)
